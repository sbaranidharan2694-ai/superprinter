## What each competitor does that we don't

### Rathna Offset — Press inventory transparency
- `/infrastructure` page lists exact machine counts: *"two 8-colour presses, two 5-colour presses and two 4-colour presses... Twin heat-set web units... Two digital production presses, one single colour and one four colour."*
- `/products` lists publisher-specific output types (Broadsheet Supplements, Tabloid Supplements, Coffee Table Books, Variable Data) — long-tail keywords
- Editorial backlinks from PrintWeek India + Xerox News for their Versant 180 install — earned trust press, not directory citations
- 200-person workforce as a recurring trust signal in body copy

### Imprint Wings — Aged domain + enterprise logo wall
- 5 near-duplicate keyword-variant pages, each H1-optimized for one query:
  - `/printing-services-in-chennai.php`
  - `/offset-printing-press-in-chennai.php`
  - `/digital-printing-press-in-chennai.php`
  - `/printing-company-chennai.php`
  - `/offset-digital-printing-facilities-chennai.php`
- 12-16 keyword-anchored internal links per page (every page reinforces every other)
- 29 enterprise client logos: Ashok Leyland, MRF, Havells, Castrol, TAFE, Indian Railway, Murugappa, Ramco, Penna Cements, National Safety Council, Asco Numatics, Terumo-Penol, Neenah
- Machine inventory page with model numbers (Mitsubishi 1G, Heidelberg KORD, ADAST 514, Oliver Sakurai 715, Robocut)
- 4.9★ / 90 Justdial reviews

### Madras Prints — Online-shop UX
- OpenCart-style shell: Add to Cart, Cart, Checkout, My Account, Wish List, Affiliate, Gift Certificates
- **Price-in-title pattern**: titles begin with the price itself
  - "Sqft Rs.6/- Flex Banner Printing in Chennai"
  - "Photo Mugs Rs.150/- | Coffee Mugs..."
- Hard-coded prices per SKU on every product tile (₹6/sqft Flex, ₹60 Backlit, ₹85 UV Flatbed, ₹1,050 Rollup, ₹250 Self-Ink Stamp, ₹15 Transparent Visiting Card, ₹48 ID Card+Lanyard)
- Service-area URL matrix: `/<service>-printing-in-<suburb>` (e.g., `/flex-printing-madipakkam`, `/best-flex-printing-in-madipakkam`)
- "Printing @ Wholesale Price" masthead
